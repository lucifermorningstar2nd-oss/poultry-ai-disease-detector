🐔 Poultry AI Disease Detection System
🌍 Turning a Personal Problem into an Technological Solution for Farmers in Ghana
This project is an AI-powered system that helps detect diseases in chickens using image recognition. It was built using a TensorFlow Lite model and a Flask backend, combined with a simple web interface that allows farmers to upload images and receive fast predictions with confidence scores.

💡 Why I Built This Project
I grew up around poultry farming because my family owned a small poultry farm in Ghana.

While helping on the farm, I noticed a serious problem:

Many farmers were unable to detect diseases in chickens early enough.

Most of the time, they had to wait for veterinary officers, but in many rural areas, vets are not always available or come too late. By the time help arrives, many birds are already sick or dead.

This situation leads to:

High loss of chickens
Financial damage to farmers
Reduced food supply
Stress and uncertainty for small farmers
🧠 The Problem I Wanted to Solve
In Ghana and many parts of Africa, poultry farmers often face:

Limited access to veterinary services
Late detection of diseases like Newcastle disease and Avian Influenza
High mortality rates in farms
Financial losses that affect families and small businesses
Many farmers lose a large part of their income because they cannot detect disease early enough.

🚀 My Solution
I decided to build an AI system that can help solve this problem.

This system allows farmers to:

Take a picture of a chicken

Upload it to the system

Get an instant prediction:

Healthy 🟢
Newcastle Disease 🔴
Avian Influenza ⚠️
It also shows a confidence score so farmers understand how certain the AI is.

🛠️ How It Works
The frontend is built using HTML, CSS, and JavaScript
The backend is built using Flask
The AI model is a TensorFlow Lite model
Images are sent to the backend using an API
The model processes the image and returns a prediction
📊 Real-World Impact
Poultry farming is very important in Ghana’s economy and food system.

Based on research, disease outbreaks in poultry can cause millions of dollars in losses and reduce food production significantly. Early detection can:

Reduce chicken mortality rates
Save farmers money
Improve food security
Increase productivity in rural farming communities
If a system like this is improved and deployed widely across Ghana, it could help reduce large-scale economic losses in the poultry industry and support small farmers who form the backbone of local food production.
